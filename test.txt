* FAISS : 

Tạo Index (Offline): Bạn chạy file dataset_scripts/build_faiss_index.py để quét toàn bộ dataset cũ, trích xuất đặc điểm và lưu vào file index của FAISS.
Truy vấn (Online): Trong file backend/routes/similar_cases.py, khi có yêu cầu /api/similar/find, FAISS sẽ mở file index đó lên và thực hiện lệnh search_similar để trả về danh sách các ca giống nhất cho Frontend hiển thị.
Tóm lại: FAISS giúp hệ thống của bạn có khả năng "so sánh đối chiếu" thông minh như một bác sĩ giàu kinh nghiệm, người đã từng xem qua hàng ngàn tấm phim MRI trong đời.
